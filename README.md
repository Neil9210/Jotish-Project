# 🔮 Jotish Project

A modern astrology-themed web application that provides users with an interactive and visually appealing experience. The project focuses on clean UI, fast performance, and responsive design.

## 🚀 Live Demo

👉 https://jotish-project-ivory.vercel.app/

---

## 📌 Features

- ✨ Modern and responsive UI
- 🔮 Astrology-themed interface
- ⚡ Fast performance with Vite
- 🎨 Styled using Tailwind CSS
- 📱 Mobile-friendly design
- 🧩 Reusable React components

---

**Frontend**
- React
- TypeScript
- Vite

**Styling**
- Tailwind CSS

**Tooling**

- PostCSS
